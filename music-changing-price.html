🎵 음반 가격 변경 자동화 프로그램

1단계: 데이터 추출 | 2단계: VLOOKUP 매칭

1 1단계: 데이터 추출 및 다운로드

원본 Excel 파일을 업로드해서 원하는 열만 추출합니다.
📋 작업순서:
1️⃣ 파일 선택 → <거래처 발송 단가 인상 리스트> 파일 업로드
2️⃣ 헤더 행 설정 → 해당 행 번호 입력
3️⃣ 상품코드 & 출고가 → 드롭다운에서 선택
4️⃣ 부가세/할인 → 필요시 설정
5️⃣ 추가 조건 설정 → 필요시 추가
6️⃣ 미리보기 → 데이터 확인
7️⃣ 다운로드 → B파일로 저장

⚙️ 출고가 설정

📝 예시: 부가세 O + 5% 할인 → 출고가 × 1.1 × 0.95

📋 조건 추가 (열 추가)

💡 텍스트 입력 = 다운로드 시 헤더명 / 필터링할 열 = 가져올 데이터 열

2 2단계: VLOOKUP 매칭

A 파일 (통유 상품조회(음반) 파일) + B 파일 (1단계 다운로드 파일)을 매칭합니다.
A의 상품코드(13자리)를 기준으로 B 파일에서 정보를 가져옵니다.
📋 작업순서:
1️⃣ A 파일 업로드 → 통유 상품조회(음반) 파일
2️⃣ B 파일 업로드 → 1단계 다운로드 파일
3️⃣ 매칭 실행 → 🔗 매칭 실행 버튼 클릭
4️⃣ 미리보기 → 데이터 확인
5️⃣ 다운로드 → 최종 결과 저장
